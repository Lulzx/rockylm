# RockyLM Chat Dataset

Synthetic single-turn conversations between a human and **Rocky**, the Eridian
alien from *Project Hail Mary*. Rocky speaks in his translated broken English:
the **"question?"** tag, triple-word emphasis (**"good good good"**),
third-person **"rocky"**/**"grace"**, dropped articles, and blunt cause→effect
science.

Generated by template composition across 46 topics (greeting, astrophage,
predator, building/Xenonite, his blindness/sound-sense, mate, crew, sacrifice,
idioms he can't parse, danger, fishing, naming, …). See `STYLE.md` for the spec.

## Format

```json
{"input": "what is astrophage", "output": "astrophage eat star heat. star get cold. star sick.", "category": "science"}
```

| field | type | meaning |
|---|---|---|
| `input` | string | the human's message |
| `output` | string | Rocky's response |
| `category` | string | topic generator |
